JAVA Exercises MERN MIT Week 2

Simple Addition Exercise

To run the sime addition function, type in add(x,y) while the x and y should be replaced by the numbers you would like to sum.

Greeting Exercise

To run the Greeting exercise, simply type greet("The Persons Name") while using a real persons name in place of the persons name.

Type of Exercise

In this exercise we can make sure that all given parameters have the same data type. To run the function, type isString( a, b, c, "string" / "number" / "boolean" / "undefined" / "null") while entering the data in place of a,b & c and define the datatype all three values should have.

Old enough to Vote Exercise

To run the function, simply type isOldEnoughToVote(age) and replace age with the age you want to test the function with. The Minimum age to vote is 18 years.

Counter Exercise

You can run the function loop(counter) to count from 0 -> 20

Simple Timer Exercise

Using the function sayHelloLate("The Persons Name", delay) we implemented a simple timer function. Replace the Person name with anyone you like and use milliseconds for the delay.

Simple Interval Exercise

Using the function sayHelloInterval("The Persons Name", interval) we implemented a simple interval function. Replace the Person name with anyone you like and use milliseconds for the interval timer. To stop the function from running, refresh the page.

Count Until Exercise

the count Until Exersise can be called by countUntil(counter,interval) while the counter defines the point when the function stops and the interval defining the time betwen the function calls.

Move in Time and Space Exercise

In this exercise we are moving in time and space using the function moveInTimeSpace(position,velocity,interval) where you define the starting position, the velocity you want to move your object as well as the interval/speed in the parameters.
